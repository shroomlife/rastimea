# oerwi
the best time tracking app ever! track your work, activities or what ever you wish. no ads, no bullshit.

## Getting Started
App URL: https://rastimea.web.app

## Installing

### With Yarn

```shell
yarn install
yarn start
```

### With NPM

```shell
npm install
npm start
```

Go to http://localhost:3000 and start hacking!
